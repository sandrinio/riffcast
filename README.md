# RiffCast

> Plug a design skill into any AI agent — produce finished animations, decks, and infographics from a one-sentence brief. Code-driven, no GUI.

<video src="demos/meta-demo/out/meta-demo.mp4" autoplay muted loop playsinline width="100%"></video>

## What it does

A user types one sentence into their preferred AI agent. RiffCast hands the agent a deterministic pipeline (brief → JSX → render → review → export) plus a curated set of design vocabularies. The agent authors the artifact; RiffCast is the rails.

The agent stays in the creative loop the whole time — RiffCast is not a black box that hides the design from the agent. That's the **thin-tools** philosophy: the agent is the designer, the tool is the runway.

## Install

The agent installs RiffCast for you. Paste this into Claude Code, Cursor, OpenCode, Trae, Claude Desktop, or any other AI agent that can read URLs:

> Install RiffCast from https://riffcast.soula.ge/INSTALL.md

The agent identifies its runtime, picks the right connector (markdown skill / MCP server / pydantic / CLI), and runs the install for you. See [INSTALL.md](./INSTALL.md) for the full protocol.

If you'd rather install manually:

```bash
# Markdown skill (Claude Code, Cursor, OpenCode, Trae)
npx skills add @riffcast/skill

# MCP server (Claude Desktop)
# Add to your mcp.json — see INSTALL.md §2B for the config block

# CLI (any shell)
npm i -g @riffcast/cli
riffcast --version
```

## Architecture

Five connectors over one core engine:

```
markdown skill (skills.sh) ─┐
MCP server ─────────────────┤
pydantic AI tool ───────────┼─→  @riffcast/core  ─→  Remotion / SVG / Slidev / HTML
CLI (riffcast) ─────────────┤
library (@riffcast/core) ───┘
```

Each connector is a thin adapter; none of them re-implement design logic. Pick the one that matches your runtime; the underlying behavior is identical.
